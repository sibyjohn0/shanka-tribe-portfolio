# Shanka Tribe Portfolio

A modern, elegant portfolio for Shanka Tribe — an organic trance and world music band from Kerala, India.

## Features

- **Responsive Design**: Works seamlessly on all devices
- **Typography Pairing**: EB Garamond headings + Inter body text for editorial elegance
- **Full Media**: Logo, band member photos, album covers, and press assets
- **Analytics**: Spotify and YouTube insights integrated
- **Mobile-First**: Clean, accessible layout

## Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/sibyjohn0/shanka-tribe-portfolio.git
   cd shanka-tribe-portfolio
   ```

2. Open `index.html` in your browser or serve locally:
   ```bash
   python3 -m http.server 8000
   ```

3. Visit `http://localhost:8000`
